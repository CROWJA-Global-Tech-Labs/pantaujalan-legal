Kebijakan Privasi (Bahasa Indonesia)

Terakhir diperbarui: 20 April 2026 · Versi aplikasi: 1.0.2

Ringkasan singkat. PantauJalan tidak memiliki server backend. Aplikasi ini hanya menampilkan feed CCTV publik milik pemerintah Indonesia. Tidak ada akun, tidak ada pelacakan pribadi, dan data yang di-cache hanya disimpan di perangkatmu.

PantauJalan (dikembangkan oleh CROWJA Global Tech Labs) adalah aplikasi Android yang menampilkan feed CCTV publik yang sudah dipublikasikan secara terbuka oleh Dinas Perhubungan dan instansi pemerintah di Indonesia. Dokumen ini menjelaskan data apa yang ditangani oleh aplikasi.

1. Data yang Kami Kumpulkan

Kami tidak memiliki server backend. PantauJalan tidak mengumpulkan, menyimpan, atau mengirimkan data pribadi kamu ke server milik kami. Daftar feed bawaan diambil sekali dari GitHub Gist publik yang berisi file terenkripsi — permintaan tersebut tidak menyertakan identitas pengguna.

Data yang disimpan di perangkat (SharedPreferences dan cache internal aplikasi):

  • Feed CCTV yang kamu tambahkan sendiri (URL, nama, tipe).
  • Daftar feed bawaan yang kamu sembunyikan.
  • Cache preview thumbnail CCTV (disimpan permanen sampai kamu hapus data aplikasi — thumbnail membuktikan stream pernah hidup, tidak dianggap basi).
  • Cache daftar kamera dan URL stream per feed (TTL ~24 jam).
  • Flag keberadaan audio per URL (dipakai ikon speaker di tile).
  • Rekaman/tangkapan layar kamera yang kamu simpan manual (disimpan di galeri perangkat via MediaStore — bukan di server kami).

Menghapus aplikasi (atau "Clear data" di Setelan Android) akan menghapus semua cache di atas. File tangkapan layar yang sudah disimpan ke galeri tetap ada karena dikelola oleh sistem, bukan aplikasi.

2. Akses Jaringan

Saat kamu membuka sebuah feed, aplikasi melakukan permintaan HTTP/HTTPS langsung ke server portal pemerintah yang bersangkutan (misalnya cctv.jogjakota.go.id, atcs-dishub.bandung.go.id, cctv.salatiga.go.id, opencctv.org, dan lain-lain). Data lalu-lintas CCTV mengalir dari server mereka langsung ke perangkatmu. PantauJalan tidak berfungsi sebagai proxy dan tidak menyimpan isi video.

Satu-satunya permintaan ke pihak ketiga selain portal CCTV adalah fetch berkala file konfigurasi terenkripsi dari GitHub Gist (gist.githubusercontent.com) untuk memperbarui daftar feed bawaan ketika kami menambahkan sumber baru.

3. Iklan (AdMob)

Aplikasi menampilkan iklan dari Google AdMob. AdMob SDK dapat mengumpulkan Advertising ID, alamat IP, informasi perangkat, dan interaksi iklan sesuai kebijakan Google. Kebijakan privasi AdMob: policies.google.com/privacy. Kamu bisa me-reset atau menonaktifkan Advertising ID lewat Setelan Android → Privasi → Iklan.

4. Izin Android

  • INTERNET — wajib untuk mengakses feed CCTV dan iklan.
  • ACCESS_NETWORK_STATE — cek koneksi sebelum fetch.
  • ACCESS_COARSE_LOCATION — opsional. Hanya dipakai saat fitur Capture/Record untuk menandai metadata EXIF/lokasi foto/video hasil tangkapan. Digunakan hanya ketika feed CCTV tidak memiliki koordinat sendiri. Lokasi tidak pernah dikirim ke server kami. Kamu bisa menolak izin ini dan aplikasi tetap berfungsi normal — hasil tangkapan hanya akan tanpa geotag.

Aplikasi tidak meminta izin kamera, mikrofon, kontak, atau lokasi presisi (FINE_LOCATION).

5. Anak-anak

Aplikasi ini ditargetkan untuk pengguna umum (13+). PantauJalan tidak secara sadar mengumpulkan informasi dari anak di bawah 13 tahun.

6. Open Source

Kode sumber aplikasi tersedia publik di github.com/CROWJA-Global-Tech-Labs/PantauJalan. Kamu bisa mengaudit sendiri permintaan jaringan dan penanganan data yang dilakukan aplikasi.

7. Perubahan Kebijakan

Kami dapat memperbarui dokumen ini sewaktu-waktu. Tanggal "Terakhir diperbarui" di atas akan ikut berubah. Perubahan besar akan diumumkan melalui release notes di halaman aplikasi.

Privacy Policy (English)

Last updated: 20 April 2026 · App version: 1.0.2

TL;DR. PantauJalan has no backend. It only shows public Indonesian government CCTV feeds. No accounts, no personal tracking, and all cached data stays on your device.

PantauJalan (built by CROWJA Global Tech Labs) is an Android app that displays publicly published CCTV feeds from Indonesian transportation agencies. This document explains what data the app handles.

1. Data We Collect

We run no backend. PantauJalan never collects, stores, or transmits your personal data to our own servers. The bundled feed list is fetched once from a public, encrypted GitHub Gist — that request does not carry any user identifier.

The app stores the following on your device only (SharedPreferences and internal app cache):

  • CCTV feeds you added yourself (URL, name, type).
  • List of bundled feeds you chose to hide.
  • Cached preview thumbnails (persisted permanently until you clear app data — a thumbnail that proved the stream existed is never treated as stale).
  • Resolved camera lists and stream URLs per feed (~24-hour TTL).
  • Per-URL audio-presence flag (drives the speaker chip on the tile).
  • Screenshots / video captures you explicitly saved (written to the device gallery via MediaStore — not to our servers).

Uninstalling the app (or "Clear data" in Android Settings) wipes every cache above. Screenshots already saved to your gallery stay because the system, not the app, owns them.

2. Network Access

When you open a feed, the app makes HTTP/HTTPS requests directly to the relevant government portal (e.g. cctv.jogjakota.go.id, atcs-dishub.bandung.go.id, cctv.salatiga.go.id, opencctv.org, and others). Video traffic flows from their servers to your device. PantauJalan is not a proxy and does not record or mirror stream content.

The only third-party request beyond the CCTV portals is a periodic fetch of an encrypted config file from a public GitHub Gist (gist.githubusercontent.com) to refresh the built-in feed list when we add new sources.

3. Ads (AdMob)

The app displays ads from Google AdMob. The AdMob SDK may collect advertising ID, IP address, device info, and ad interactions per Google's policies. AdMob privacy policy: policies.google.com/privacy. You can reset or opt out of the Advertising ID via Android Settings → Privacy → Ads.

4. Android Permissions

  • INTERNET — required to fetch CCTV streams and ads.
  • ACCESS_NETWORK_STATE — connectivity check before fetches.
  • ACCESS_COARSE_LOCATION — optional. Used only by the Capture/Record feature to tag EXIF/location metadata on saved photos and videos, and only when the CCTV feed itself does not carry coordinates. Location never leaves your device. You can deny this permission and the app still works normally — captures will simply be saved without geotags.

The app does not request camera, microphone, contacts, or fine-location (FINE_LOCATION) permissions.

5. Children

PantauJalan targets general audiences (13+). It does not knowingly collect information from children under 13.

6. Open Source

The app source is public at github.com/CROWJA-Global-Tech-Labs/PantauJalan. You can audit the network requests and data handling yourself.

7. Changes

We may update this document from time to time. The "Last updated" date above will change accordingly. Material changes will be announced in the app's release notes.
